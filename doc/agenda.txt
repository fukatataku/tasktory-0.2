07. 作業時間表示位置変更
    現在は以下のような表示にしている
    ====
    /path/to/task @?? [mm:dd-mm:dd]
     # COMMENT
    ====
    これには以下の問題点がある
    - 行が長くなりがち
    - ジャーナル全体で作業時間のカバレッジが分かりにくい

    例えば以下のように変更する
    ====
    /path/to/task @??
     mm:dd-mm:dd # description
     mm:dd-mm:dd # description
     # COMMENT
    ====
    これなら行が長くなることもなく、時間ごとに何をしていたかも記録できる
    登録されている時間のカラムが統一されているので、全体も把握しやすい
    ただし、ジャーナル全体が縦に長くなる欠点もある
